# Pycelium Website

Official website for Pycelium — building FOSS packages for distributed applications.

## About

Pycelium is a portmanteau of "Python" and "mycelium." Like fungal mycelium networks that grow without central coordination, our vision is distributed applications that run autonomously on two or more nodes.

Our mission is to decentralize data ownership and information flow, enabling maximally decentralized applications without the typical Web3 crypto grift.

Required reading: [Cypherpunk Manifesto](https://www.activism.net/cypherpunk/manifesto.html)

## Tech Stack

- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Post-processor:** PostCSS + Autoprefixer

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
