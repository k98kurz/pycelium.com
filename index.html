Project Spotlight

micropycelium

Micropycelium is a routable mesh network programmed in micropython and currently compatible with ESP32 using ESP-NOW. It creates and maintains a spanning tree of nodes, then it uses greedy routing based upon the VOUTE (Virtual Overlays Using Tree Embeddings) system.
